# iotex-dapp-sample
This repo provides a boilerplate IoTeX Dapp, with examples to use the antenna SDK, connect to the ioPay wallet, show balance, send IOTX and send XRC20 tokens.

## How to Use

Install it and run:

```bash
npm install
npm run dev
```
